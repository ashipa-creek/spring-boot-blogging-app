# Spring Boot Blogging Application

A full-stack blogging application built using Spring Boot and Thymeleaf.  
Users can create posts, add comments, and manage blog content.

## Features
- Create and view blog posts
- Add and delete comments
- Thymeleaf-based UI
- Spring Boot MVC architecture
- MySQL integration
- CRUD operations

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- Thymeleaf
- MySQL
- Maven
